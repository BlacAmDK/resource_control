# 服务器资源占用程序

为了保持服务器的CPU与内存使用率，通过此程序在不影响服务器其他程序运行的情况下将CPU与内存占用率控制在一定范围内。
目前的运行逻辑：
1. 根据CPU核心数，在每个核心各建立一个线程，每个线程根据当前核心使用率以200ms为单位，通过选择计算和休眠两种状态将CPU使用率保持在55%左右。
2. 内存以200ms为单位循环计算内存使用率，如果低于45%，则申请并占用1%内存(每次最大申请1G)；在45%到55%之间时不做任何操作；高于55%时，根据内存使用率释放占用的内存使其尽量达到50%。

运行方式：
直接添加执行权限放到后台运行，如：

```bash
nohup ./resource_control &
```
